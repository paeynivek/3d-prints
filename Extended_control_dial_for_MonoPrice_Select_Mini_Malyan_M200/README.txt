# Summary

This is an adoption of the Better Dial http://www.thingiverse.com/thing:1677768

I modified the Better Dial to be used with the original dial to give it a more polished look. It still sticks out plenty to grab the sides for more control but can still spin on the face of the dial as well. The original dial snaps on snuggly with no need for gluing. 

I used clear PLA filament printed with 10% infill, it gives a nice back light from the blue LED lights.

UPDATE: I reduced the depth of the bottom hole so less likely to bottom out.
